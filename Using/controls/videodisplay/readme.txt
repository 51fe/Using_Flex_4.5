For most of the VideoDisplay examples, you need to server the MyVideo.flv file from a web server. 

For VideoDisplayCamera.mxml, you need to connect a USB camera to your machine.

For VideoDisplayFMS.mxml, you need to do the following:

    - install FMS 2
    - create the following dir structure under FMS:
        Flash Media Server 2\applications\videodisplay
        Flash Media Server 2\applications\videodisplay\scripts
        Flash Media Server 2\applications\videodisplay\streams
        Flash Media Server 2\applications\videodisplay\streams\_definst_
    - Copy the file assets\bike.flv to _definst_
